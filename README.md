# The One Ring Dice Roller

Made with React + TypeScript + Vite.

## Creating a new build

- Create a new `dist` branch from main
  - `git checkout main`
  - `git branch -D dist`
  - `git checkout -b dist`
- Create a new build
  - `npm run build`
- Commit and push
  - `git add docs`
  - `git commit -m Deploy`
  - `git push -f origin dist`
